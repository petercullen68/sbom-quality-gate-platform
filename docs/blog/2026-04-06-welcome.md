Welcome to the SBOM Quality Gate documentation site!

SBOM Quality Gate is a centralized Software Bill of Materials validation and quality orchestration platform. We built it to solve a real problem: the fragmented SBOM tooling ecosystem.

<!-- truncate -->

## The Problem

Organizations adopting SBOMs for compliance (NIS2, FDA, executive orders) face tool sprawl:

- Quality scoring tools
- Vulnerability scanners  
- License compliance checkers
- Format validators

Each tool has its own API, its own output format, and its own integration requirements. This creates friction and scattered results.

## The Solution

SBOM Quality Gate acts as a single point of ingestion. Submit your SBOM once, and the platform:

1. Validates against schema specifications
2. Scores quality using configurable profiles
3. Routes to downstream tools
4. Aggregates results into a unified view

## What's Available Now

- **CycloneDX and SPDX support**: Both major formats are recognized and validated
- **Quality scoring via sbomqs**: Comprehensive quality assessment out of the box
- **Async processing**: Upload fast, process in the background
- **RESTful API**: Easy CI/CD integration

## What's Coming

- **Validation profiles**: Custom pass/fail criteria per team or product
- **Dependency-Track integration**: Vulnerability scanning alongside quality
- **API authentication**: API keys and access control
- **Result webhooks**: Push notifications when validation completes
